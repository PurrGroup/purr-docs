## 文章表

| 字段名         | 数据类型         | 含义/备注                        |
| -------------- | ---------------- | -------------------------------- |
| ID             | BIGINT UNSIGEND  | 文章索引                         |

| share_count    | INTEGER UNSIGEND |                                  |

| status         | TINYINT UNSIGEND | 公开/草稿 0/1                    |

| ping_status    | VARCHAR(20)      |                                  |
| to_ping        | TEXT             |                                  |
| pinged         | TEXT             |                                  |
| is_original    | TINYINT UNSIGEND | 原创/转载 0/1                    |
